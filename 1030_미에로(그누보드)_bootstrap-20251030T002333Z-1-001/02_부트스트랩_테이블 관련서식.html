02_부트스트랩_테이블 관련서식

테이블 태그 : table, caption, thead(표상단), tr(줄), td(칸), th(제목칸), tbody(표본문)로 구성됨

셀 병합 : 칸 병합(colspan), 줄병합(rowspan)

표(table) 관련서식

  • class="table" : 테이블 기본 서식이 적용됨.
  • class="table table-striped" : 각 행마다 홀수/짝수 번째 색상이 다르게 적용
  • class="table table-bordered" : 테이블 태그에 테두리 적용
  • class="table table-hover" : 테이블에 마우스 호버시 색상전환
  • thead class="table-dark" : 테이블에 마우스 호버시 색상전환
  • tr class="text-center" : 제목 태그 가운데 정렬
  • class="table-success" : bg-color #d1e7dd 셀배경 초록색
  • class="table-danger" : 셀배경 빨강색
  • class="table-info" : 셀배경 하늘색
  • class="table-primary" : 셀배경 파랑색
  • class="table-responsive" : 화면의 너비가 768px미마인 경우 가로로 스크롤 됨
