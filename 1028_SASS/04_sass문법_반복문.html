1. @for 문

@for 키워드는 '스타일을 반복적'으로 출력하는데 사용됩니다.

자바스크립트의 for문과 유사합니다. 

    문법1)
    through : 종료만큼 반복
    @for $변수 from 시작 through 종료 {
      //반복할 내용
    }

    문법2)
    to : 종료전까지 반복
    @for $변수 from 시작 to 종료 {
      //반복할 내용
    }

$변수 => #{변수}

#{변수}는 코드 어디든지 변수 값을 사용 할 수 있습니다.

적용예) $img:unquote('./img/#{$img}');

h1{background-image:url('./img/#{$img};')}
